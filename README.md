# Automobile Price Prediction using Machine Learning

## Project Overview

This project develops machine learning models to predict automobile prices using the IBM Automobile Dataset. The study applies exploratory data analysis, data cleaning, feature engineering, and predictive modelling techniques to identify the most influential factors affecting vehicle prices.

## Dataset

* Dataset: IBM Automobile Dataset
* Records: 205 vehicles
* Features: 26 original attributes
* Target Variable: Price

## Techniques Used

### Data Preprocessing

* Missing value treatment
* Mean and mode imputation
* Feature normalisation
* One-hot encoding

### Feature Engineering

* City fuel consumption (L/100km)
* Horsepower categorisation
* Vehicle volume
* Power-to-weight ratio
* Average MPG
* Horsepower per engine size

### Machine Learning Models

* Linear Regression
* Random Forest Regression
* XGBoost Regression

## Results

| Model             | R² Score |
| ----------------- | -------- |
| Linear Regression | 0.8596   |
| Random Forest     | 0.9443   |
| XGBoost           | 0.9292   |

Random Forest achieved the highest predictive accuracy.
